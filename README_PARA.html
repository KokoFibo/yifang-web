<Para /> Component

Modern Astro component for rendering paragraphs and lists with dynamic text styling (bold, italic, underline), default styles, and font size control.

Features

  • Default italic / bold / underline
  • Font size control
  • Styled keywords & phrases
  • Ordered & unordered lists
  • Tailwind CSS safe

Import

import Para from "@/components/Para.astro";

Basic Usage

<Para
  main={["Hello World"]}
  defaultStyle={{ italic: true, fontSize: "lg" }}
/>

Props

PropTypeDescription
mainstring[]Main paragraphs
liststring[]Optional list items
listStyleul | olList type
defaultStyleobjectDefault text style
thestylearrayKeyword styling rules

Example

<Para
  main={["Yifang Academy provides Recruitment Solutions."]}
  list={["Job Posting", "Screening", "Onboarding"]}
  listStyle="ul"
  defaultStyle={{ fontSize: "lg" }}
  thestyle={[
    { words: ["Yifang Academy"], styles: ["bold"] },
    { words: ["Recruitment Solutions"], styles: ["underline"] }
  ]}
/>
